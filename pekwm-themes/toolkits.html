Toolkits

To help with finding matching toolkit themes for the pekwm themes listed on this site a few recommended toolkit themes are listed here.

Gtk+3

NextStep Look

OneStepBack is a NextStep inspired theme for Gtk+2 and Gtk+3.

CDE Look

Belle Pintos Grande is a CDE inspired theme for Gtk+2 and Gtk+3.

Gtk+2 Look

Raleigh Reloaded is a Gtk+3 version of the old Gtk+2 default theme.

Backwater is similar to Raleigh and includes several color variants.

Blueshell is a Gtk+3 port of the well-known Bluecurve theme from Red Hat.

Clearlooks Phenix is a Gtk+3 port of the classic Clearlooks theme.

Clearlooks Overcast is a grey variation of the Clearlooks theme.
